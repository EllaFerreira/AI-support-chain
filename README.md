## Getting Started

1. Clone the repository:

```
   git clone 
```

2. Install dependencies:

```
 npm install
```

3. Run the Docker container:
  
  ```
  docker-compose up
  ```

## Running the app

```bash
npm start
```

The app will be accessible at `http://localhost:3000` üöÄ
